## Project Overview

This project focuses on analyzing global air quality data and applying Machine Learning and Reinforcement Learning techniques to understand pollution patterns and support intelligent traffic-control decisions. The project combines Exploratory Data Analysis (EDA), Classification, Clustering, Dimensionality Reduction, and Reinforcement Learning to provide meaningful insights into Air Quality Index (AQI) trends.

---

## Objectives

* Analyze and visualize global air quality data.
* Identify key pollutants affecting AQI levels.
* Build machine learning models for AQI classification.
* Perform clustering to discover pollution patterns.
* Apply Principal Component Analysis (PCA) for dimensionality reduction.
* Develop a Reinforcement Learning agent for AQI-based traffic control decisions.

---

## Dataset

**Dataset Name:** Global Air Quality Dataset

The dataset contains:

* Air Quality Index (AQI) values
* PM2.5 and PM10 concentrations
* Carbon Monoxide (CO)
* Nitrogen Dioxide (NO₂)
* Ozone (O₃)
* Sulfur Dioxide (SO₂)
* Temperature
* Humidity
* Wind Speed
* City-based environmental observations
